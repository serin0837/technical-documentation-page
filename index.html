Fundamental

what is computer?

  • 1930s has taken off machines that can perform calculations and operations on data.
  • hardware : physical stuff like keyboard
  • software: systems take input and give you output (not physical stuff) the set of instructions written to be processed(01001 stuff) and executed by the hardware that makes up a computer

What is operating system?

  • Operating system is programs that manage with the different software
  • Operating systems are essentially master programs that manage and co-ordinate the different software and how they get access to the physical hardware of a computer.
  • Unix, DOS

Terminal & Shell & Bash

  • A console is a set of tools that are used to operate a machine.
  • A terminal allows users to input commands but this application doesn't know what to do with the command itself.
  • A shell is a command line interpreter( interprets commands and executes a program)
  • Bash is both a command line interpreter and a programming language.
  • command: ls (list directory contents) :which folders and directories are currently inside of the current directory. cd (change directory), also go to home directory start . . (open directory) mkdir (make directory) rmdir (delete directory) touch (make file) rm (remove files and directories)

Version Control

version control

  • git is version control software and some of its most basic features will allow us to fulfill all the requirements of a basic content timeline.
  • version control system is software that helps you control (or manage) the different version.
  • ex) git, subversion, mercurial

git and version control

  • repository
  • working directory
  • staging area
  • SHA
  • branch: A branch is when a new line of development is created that diverges from the main line of development.
  • The branch that is created by default when a new empty git repository is created is the master branch. Whenever a new git commit is created then master branch is updatd so it points to the most recent commit.
